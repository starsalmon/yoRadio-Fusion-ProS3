## Plugins (this fork)

Put custom plugins under `src/plugins/<MyPlugin>/`.

The plugin API is described in `src/pluginsManager/README.md` (mostly upstream text, but still accurate for the hook points used here).

Notes:
- This repo’s code lives under `src/…` (some upstream docs reference `yoRadio/src/…`).
- Existing “first-party” plugins (e.g. backlight/ledstrip) live under `src/plugins/`.
